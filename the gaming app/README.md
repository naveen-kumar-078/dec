# PlayZone - The Ultimate Gaming Marketplace

Welcome to PlayZone, a premium, production-ready gaming marketplace web application.

## Features
- **Immersive 3D Background**: Interactive 3D floating geometries using Three.js.
- **Modern UI/UX**: Glassmorphism, Neon aesthetics, and smooth animations.
- **Game Library**: Browsable grid of top games with hover effects.
- **Detailed Game Views**: Rich modal interface for game details.
- **Checkout Flow**: Simulated purchase process with multiple payment options.

## Setup & Running
1.  **Requirements**: valid internet connection (to load Three.js and Fonts from CDNs).
2.  **Run**: simply open `index.html` in any modern web browser.
    - Double click `index.html` in your file explorer.

## Technologies Used
- HTML5
- CSS3 (Variables, Flexbox, Grid, Animations)
- JavaScript (ES6+)
- Three.js (via CDN)
- FontAwesome (Icons)
- Google Fonts (Outfit, Rajdhani)
